# Stack-Overflowers-Sibathon26
SIBATHON 2026: AI in business - AI Marketing ROI Predictor


Problem Statement
Small and medium businesses often invest in multiple marketing campaigns without knowing which ones will yield the best results. This leads to wasted budgets, missed opportunities, and difficulty in making data-driven decisions. There is a need for a system that can analyze past marketing performance and provide actionable predictions for future campaigns, helping businesses optimize their marketing investments.
Proposed Solution
The AI Marketing ROI Predictor is an intelligent tool designed to assist businesses in making smarter, data-driven marketing decisions. By analyzing historical campaign data, the system predicts the potential outcomes of future campaigns and provides recommendations to maximize return on investment (ROI) while minimizing wasted spending.
Key Features
•	Upload historical marketing campaign data in CSV format (campaign name, amount spent, sales generated, platform).
•	AI predicts expected performance of future campaigns.
•	Provides actionable recommendations:
o	Identify campaigns with high potential ROI for increased investment.
o	Highlight campaigns likely to underperform to reduce or avoid spending.
•	Optional visualization of campaign performance trends.
•	Human-readable insights for non-technical users.
Technologies Used
•	Frontend: HTML, CSS; optional use of React for interactive dashboards
•	Backend: Python with Flask or FastAPI
•	AI/ML: scikit-learn for regression or classification models
•	Data Handling: CSV input for campaign data; sample data used for demonstration purposes
